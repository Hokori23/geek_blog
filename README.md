# npm 命令汇总

## npm run server

进入后端目录 \_server

## npm run server:install

安装后端目录下的 npm 依赖项

## npm run server:build

typescript 一次编译

## npm run server:build:watch

typescript 监听变动进行编译

## npm server:dev

开发模式下启动服务器

## npm server:serve

生产模式下启动服务器

## npm run client

进入前端目录 \_client

## npm run client:install

安装前端目录下的 npm 依赖项

## npm run client:dev

开发模式下启动 Nuxt

# 文件目录

## \_client

前端目录

## \_server

后端目录
